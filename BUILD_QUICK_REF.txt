# Quick Build Reference

## Prerequisites
```powershell
# Activate virtual environment
.\.venv\Scripts\Activate.ps1

# Install build tools
pip install pyinstaller
```

## Build Commands

### Quick Build
```powershell
# Automated (recommended)
.\build_exe.ps1

# Manual
pyinstaller pogadane.spec --clean
```

### Output
```
dist/Pogadane.exe    (~100-200 MB)
```

### Test
```powershell
.\dist\Pogadane.exe
```

## Common Issues

**"No module named 'flet'"**
→ Activate virtual environment first

**"Unable to find 'pogadane'"**
→ Run from project root directory

**Antivirus blocks exe**
→ Add exception or code-sign the executable

## Distribution

```
Pogadane-v0.1.8.zip
├── Pogadane.exe
├── README.txt
└── LICENSE.txt
```

Users need to install separately:
- faster-whisper or whisper
- Ollama (optional)
