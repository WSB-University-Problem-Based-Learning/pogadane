POGADANE - SUPER SIMPLE INSTALLATION INSTRUCTIONS
================================================================

For users who just downloaded pogadane-main.zip and want to get started FAST!

================================================================
STEP 1: Extract the ZIP file
================================================================
1. Find "pogadane-main.zip" in your Downloads folder
2. Right-click â†’ "Extract All..."
3. Extract to your Desktop or Documents

================================================================
STEP 2: Install Python Packages (ONE COMMAND!)
================================================================
1. Open the extracted "pogadane-main" folder
2. Hold SHIFT + Right-click in empty space
3. Select "Open PowerShell window here"
4. Copy and paste this ENTIRE line:

python -m pip install ttkbootstrap google-generativeai

5. Press ENTER and wait (takes 30-60 seconds)

================================================================

================================================================

STEP 4: Download Required Tools
================================================================
You need 2 files (put them in the pogadane-main folder):

A) yt-dlp.exe
   Download from: https://github.com/yt-dlp/yt-dlp/releases/latest
   Click "yt-dlp.exe" under Assets

B) faster-whisper-xxl.exe
   Download from: https://github.com/Purfview/whisper-standalone-win/releases
   Get: Faster-Whisper-XXL_r245.4_windows.7z
   Extract with 7-Zip â†’ Find faster-whisper-xxl.exe

================================================================
STEP 5: Install Ollama (For AI Summaries)
================================================================
1. Download from: https://ollama.com/
2. Install Ollama
3. Open PowerShell and run:

ollama pull gemma3:4b

(This downloads the AI model, takes 5-15 minutes)

================================================================
STEP 6: Start Pogadane!
================================================================
In PowerShell (in the pogadane-main folder):
